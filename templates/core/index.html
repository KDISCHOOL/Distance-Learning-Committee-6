Distance Learning Committee operation USER Guide

Welcome to DLC operation simple demo.

본 사이트는 교원 이메일/강좌 정보를 관리하기 위한 업무용 웹앱입니다. 아래 항목을 클릭해서 각 기능으로 이동하세요.

  • Faculty Upload (관리자) — 엑셀 파일로 Faculty DB를 업로드/병합합니다. (관리자 PIN 필요)
  • Faculty Enrich — Korean_name 리스트를 업로드하면 이메일이 병합된 엑셀을 다운로드할 수 있습니다.
  • Faculty Search — 교수님 성명으로 검색해서 이메일을 확인하고 간단하게 복사할 수 있습니다.
  • Course Upload (관리자) — Course Modality 엑셀 업로드/병합 (password, reason 등 포함 가능, 관리자 PIN 필요)
  • Course Search — 교수님은 강좌를 검색하고 온라인 수업을 신청(Apply) 또는 기존 신청 내역을 조회(Look up)를 할 수 있습니다.
  • (관리자용) 최종 정보 보기 — 관리자 PIN을 입력하면 DB에 업데이트된 내용을 엑셀로 다운로드합니다.
  • Django Admin — (사이트 관리자 전용) 레코드 직접 수정/확인용
